# GerenciadorDeSenhasSeguras
Projeto desenvolvido em Java com foco em segurança da informação. A aplicação oferece um meio seguro para o armazenamento e gerenciamento de senhas, utilizando criptografia (AES), autenticação de dois fatores (2FA), geração de senhas aleatórias e fortes, além da checagem de possíveis vazamentos por meio de uma API externa.
